# Chinese Celebration Single Page

This project is now structured for GitHub Pages.

## Structure

- `index.html` → main page (GitHub Pages entry)
- `images/home-banner.jpg` → banner image
- `images/spring-festival.jpg` → Spring Festival image
- `images/mid-autumn.jpg` → Mid-Autumn image
- `images/dragon-boat.jpg` → Dragon Boat image

## Replace images

Upload your images with these exact names, and they will work without editing HTML:

- `images/home-banner.jpg`
- `images/spring-festival.jpg`
- `images/mid-autumn.jpg`
- `images/dragon-boat.jpg`

## Publish on GitHub Pages

1. Push this repo to GitHub.
2. In GitHub: **Settings → Pages**.
3. Source: deploy from branch (`main`), folder `/ (root)`.
4. Save and open the generated site URL.
